# Emertech dMRV Landing Page

Vite + React + Tailwind CSS v4.

## Deploy (no local npm needed)
1. Create a new repo on github.com and upload these files (drag the folder contents into the "Add file > Upload files" page).
2. On vercel.com: Add New > Project > import the repo. Vercel auto-detects Vite. Click Deploy.

## Run locally (once your npm works)
```bash
npm install
npm run dev
```
